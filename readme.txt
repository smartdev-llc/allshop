How To Run:
This is simple frontend website of an ecommerce bussiness for allshop.
This will be run when open index.html file in browser.
There will be no need to install any database.
the data will store as a constent dataset.

Usage Libraries:
1. Frontend Freamword: Angularjs 1.8
2. Bootstrap theme: SB-Admin-2
3. Icon Library: Fontwasome
4. Component Library:
	1. Select: Jquery Select2
	2. Alert: Sweet Alert
